# Release: SpaceCraft SDK v0.63.1

Date: 2025-11-24

## Short description:

SpaceCraft v0.63.1 renames a few API functions, for consistency.


## Full description:

Blockchain API renamed: `epoch_start_block_*` -> `get_epoch_start_block_*`, for consistency.

Set block timestamp methods renamed to `block_timestamp_seconds` and `block_timestamp_millis`, respectively, also for consistency.

These changes were supposed to be part of v0.63.0.
